# v0 Project Instructions

This is a Fainted Studio portfolio website created with v0 by Vercel.

## Quick Start

1. Install dependencies: `npm install` (or `pnpm install`)
2. Run dev server: `npm run dev`
3. Open http://localhost:3000

## Project Information

- **Site Name:** Fainted
- **Domain:** fainted.studio
- **Contact:** matthew@fainted.studio
- **Type:** Video Editor Portfolio

## Key Sections

1. **Navigation** - Sticky header with smooth scroll links
2. **Hero** - Introduction with VSL video player
3. **Work** - Video portfolio gallery with filters
4. **Process** - 4-step creative workflow
5. **FAQ** - Interactive questions and answers
6. **Contact** - Contact form and social links
7. **Footer** - Site footer with links

## Color Scheme

- Background: #0E0E12
- Primary Text: #E6E6E6
- Secondary Text: #8A8A8A
- Accent: #FF3D81
